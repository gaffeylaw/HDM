  2. 界面操作：

    • 文件列表

      在文件列表部分，从下拉列表中选择“文件名”或“描述”，再在输入框中输入对应的关键字，然后点击enter键，可以查询对应的文件；

      在文件列表头栏中：

      点击“上传新文件”按钮，可以进入上传文件界面；

      点击“批量上传文件”界面，可以进入批量上传文件界面；

      文件列表数据列包括6列：

      1. 操作：在操作栏可以

        点击“编辑”图标进入 上传文件编辑界面;

        点击“删除”图标可以将当前文件删除；

        Note

        这两个操作由不同的权限限制，该列的操作的显示是由当前角色对应的权限来决定的；

      2. 文件名：文件的名称；

      3. 文件类型：文件的文件类型；

      4. 描述：针对该文件的描述信息；

      5. 文件大小(KB)：文件的大小；

      6. 最后更新日期：文件的最后更像日期；
