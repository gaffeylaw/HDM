文件管理-编辑

  1. 界面所需角色：

  2. 页面路径：

    从文件管理首页，在需要编辑的附件前点击“编辑”图表；

  3. 界面操作：

    • 按钮操作：

      保存：保存当前新上传的文件，页面跳转到文件管理主页；

      取消：取消当前操作，页面跳转到文件管理主页；

    • 一般信息区域:

      该部分是上次的详细填写项:

      1. 文件：选择需要上传的文件；

      2. 上传新文件：如果需要替换前面上传的文件，则在此上传新的文件；

      3. 文件类型：选择上传文件的文件类型；

      4. 描述：针对该上传文件的描述；

      5. 是否私有：如果勾选，表示该文档只能被上传者使用；
