  2. 界面操作：

    • 视图

      在调查问卷视图部分，从下拉列表中选择一个列表视图，以直接进入该列表页，或单击创建新视图，以定义您的自定义视图；要查看队列中的调查问卷，直接从下拉列表中选择该视图列表。

    • 调查问卷列表

      在调查问卷列表部分，从下拉列表中选择“问卷名称”，再在输入框中输入对应的关键字，来查询对应的问卷列表；

      在事故单列表栏中，点击新建按钮，可以进入新建调查问卷界面

      调查问卷列表数据列包括4列：

      1. 操作：在操作栏可以

        点击“答题”图标进入 调查问卷答题页面;

        点击“查看”图标进入查看调查问卷页面；

        点击“结果”图标进入查看调查问卷结果页面；

        点击“范围”图标进入调查问卷范围页面;

        Note

        这几个操作由不同的权限限制，该列的操作的显示是由当前角色对应的权限来决定的；

      2. 问卷名称：调查问卷的名称；

      3. 描述：当前问调查问卷的描述信息；

      4. 记录状态：标示当前调查问卷的状态；
